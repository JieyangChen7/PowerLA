cgeam adds/transposes matrices: C = alpha*op( A ) + beta*op( B ).

The operation supports also the following in-place transformations C = alpha*C + beta*op( B ) C = alpha*op( A ) + beta*C

Parameters
[in]transAmagma_trans_t. On entry, transA specifies the form of op( A ) to be used in the matrix multiplication as follows:
  • = MagmaNoTrans: op( A ) = A.
  • = MagmaTrans: op( A ) = A**T.
  • = MagmaConjTrans: op( A ) = A**H.
[in]transBmagma_trans_t. On entry, transB specifies the form of op( B ) to be used in the matrix multiplication as follows:
  • = MagmaNoTrans: op( B ) = B.
  • = MagmaTrans: op( B ) = B**T.
  • = MagmaConjTrans: op( B ) = B**H.
[in]mINTEGER The number of rows of the matrix op( dA ). M >= 0.
[in]nINTEGER The number of columns of the matrix op( dA ). N >= 0.
[in]alphaCOMPLEX On entry, ALPHA specifies the scalar alpha.
[in]dACOMPLEX array, dimension (LDDA,k), where k is N when transA = MagmaNoTrans, and is M otherwise.
[in]lddaINTEGER The leading dimension of the array dA. LDDA >= N when transA = MagmaNoTrans, and LDDA >= M otherwise.
[in]betaCOMPLEX On entry, BETA specifies the scalar beta.
[in]dBCOMPLEX array, dimension (LDDB,k), where k is N when transB = MagmaNoTrans, and is M otherwise.
[in]lddbINTEGER The leading dimension of the array dB. LDDB >= N when transB = MagmaNoTrans, and LDDB >= M otherwise.
[in,out]dCCOMPLEX array, dimension (LDDC,N). dC can be input, making the operation in-place, if dC is passed as one of the pointers to dA or dB. The M-by-N matrix dC.
[in]lddcINTEGER The leading dimension of the array dC. LDDC >= M.
[in]queuemagma_queue_t Queue to execute in.
